# Event-Driven Multiclient Chat Application on xv6
- The original repo: [ProjectTinyNet](https://github.com/ChenXuanting/ProjectTinyNet.git)

# Prerequire
- QEMU
- Python3

# Installation
```bash
git clone https://github.com/tadtd/xv6-multiclient-chat

make qemu
```
- Run the server before connecting.
```bash
python3 server.py
```
